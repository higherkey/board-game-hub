# Symbology Game Flow

## 1. Game Setup (Lobby)
- **Host**: Selects "Symbology".
- **Start**: Click "Start Game".

## 2. Clue Phase
- **Components**: Grid of images (Concept style).
- **View**:
    - **Host/Board**: Shows the target word (Secret).
    - **Guesser**: Waits.
    - **Clue Giver**: Selects concept icons to describe the word.
- **Action**:
    - Clue Giver clicks icons to place markers (Main Concept, Sub Concept).
    - Markers appear on Board for everyone to see.

## 3. Guessing Phase
- **View**:
    - **All**: See the board with marked icons.
    - **Guesser**: Input form to guess the word.
- **Action**:
    - Guesser submits guess.
    - If correct -> Win.
    - If wrong -> Continue / Time penalty?

## 4. End Round
- **View**: Summary.
- **Action**: **Next Round** (Rotate roles).
